# Interactive and Animated

Included natively with the package is drag and drop functionality between different card piles. Passing cards also comes animated, with multiple easy to use animations.An easy to understand system for setting up rules for passing and receiving cards makes creating card games a breeze.

# Extend Class Card

Easily extend class Card to create any type of card you would like. Tarot Card, Flash Cards, Japanese-trademarked-little-monster-cards, anything you could imagine!
